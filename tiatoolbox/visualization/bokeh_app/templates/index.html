Directions for use:

  • Select the slide at the top-right of the window. The slide name denotes the slide diagnosis.
  • By default, the glands are coloured by their node explanation value - ‘hot’ glands indicate areas contributing to the slide being predicted as abnormal.
  • Alternatively, glands can be coloured by individual features by clicking on the 'Colour glands by' box on the right.
  • These glands can be coloured using different colour maps, by selecting the ‘colourmap’ drop-down menu on the right.
  • By default, the gland-graph overlaid on the slide is hidden. This can be toggled on by selecting ‘graph’ at the bottom-right.
  • As you zoom in, smaller objects in the overlay will become visible, including different types of nuclei.
  • These objects can be toggled on and off by clicking on the appropriate names on the right and the default colours can be changed.
  • Feature explanations for each gland can be seen by hovering over individual nodes. Here, we show the top-5 features ranked in order from 1-5.

Note, this interface has not been optimised for smartphones or tablets.
